# LIVROS_VIAJANTES-PIBIEX
Livros Viajantes é um projeto desenvolvido pela acadêmica de TADS Rhaaby Alves Ferreira afim de promover a cultura através da tecnologia, usando os conhecimentos aprendidos em laboratório e aplicando-os na prática!

---

# Livros Viajantes in a nutshell:

uma aplicação WEB voltada para a troca, doação e até mesmo empréstimo de livros entre jovens da rede pública de ensino a partir dos 12 anos, com o objetivo de incentivar e criar mais leitores na região do Bico do Papagaio. 
Inspirado em outros serviços como Goodreads, Instagram e X, o usuário poderá publicar e pegar livros, interagindo através de um chat de mensagens diretas para combinar os detalhes da transação.
